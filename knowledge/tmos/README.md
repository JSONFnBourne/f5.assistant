# TMOS (BIG-IP) Content

This directory contains material specific to TMOS-based BIG-IP systems.

Suggested structure (to be refined as we grow F5KSI):

- `onboarding/` – Initial configuration, licensing, provisioning, HA setup.
- `networking/` – VLANs, Self IPs, routes, trunks, tunnels.
- `ltm/` – LTM-specific profiles, virtual servers, pools, monitors.
- `dns/` – DNS/GTM configuration, listeners, wide IPs, DNSSEC.
- `apm/` – Access policies, SSO, authentication, federation.
- `asm/` – Web application firewall policies, signatures, logging.
- `sslo/` – SSL Orchestrator topologies, services, service chains.
- `swg/` – Secure Web Gateway policies, URL filtering, reporting.

We can adjust this layout to match your actual environment and operational workflows.
