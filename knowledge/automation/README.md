# Automation

This directory is for automation assets around F5 platforms, including TMOS and F5OS.

Potential subdirectories:

- `as3/` – AS3 application declarations.
- `do/` – Declarative Onboarding (DO) declarations.
- `ts/` – Telemetry Streaming configuration.
- `ansible/` – Playbooks and roles for F5 modules.
- `terraform/` – Terraform configurations and modules.
- `ci-cd/` – Pipeline examples integrating F5 changes.

As we build out F5KSI, we can add concrete, reusable examples under each tool.
