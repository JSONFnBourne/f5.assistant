# iRules

This directory is dedicated to iRules and related artifacts:

- Reusable iRule libraries.
- Example iRules for LTM, DNS, ASM, APM, and other modules.
- Testing and validation approaches for iRules (e.g., traffic captures, logs).

Structure can be developed by use case (e.g., `security/`, `routing/`, `header-manipulation/`) or by application.
