# iRules Main Library

Workspace: irule-main
Goal: Curate and manage F5 iRules, specifically focusing on the tmsh subject-matter expert model and event order logic.

## Context
This project serves as the central repository for iRules data curation, testing, and fine-tuning pipelines.
It contains data files like `irules_https_event_order.jsonl` and scripts for chunking, grading, and generating QA pairs.

## Left Off
- The repository contains the complete `tmsh Pipeline` README and the core data files.
- The pipeline was set up for end-to-end data curation for the tmsh LLM model.
